Завдання: Дано дійсне число a. Не використовуючи тимчасові змінні 
та користуючись тільки операціями зсуву, отримати значення: а*16,а*1024 
– за одну операцію.
Код:
/* 
Int main - некая функция которая срабатывает при запуске программы 
Return 0 - после окончания функции сигнализирует об окончании программы 
{- тело функции все сто между ними 
; - говорит о том, что строка кода завершена 
%-значение, что помогает выводить переменную 
d - didjest - тоесть цифры, которые мы берём из значения int 
/n - переход на новую строку 
For — цикл с предусловием. Это значит, что сначала проверяется условие, 
и если оно true, то дело выполняется.
<=  - меньше или равно заданному числу, в нашем случае 10
++ - это оператор увелечиния значения на 1, а -- наоборот 
<<- сдвиг влево

*/

#include <stdio.h>

int main() {

    int num=8, i;

    for (i = 0; i <= 10; ++i) {
        printf("Left shift by %d: %d\n", i, num << i);    
    }

    return 0;
}
